Project Overview

Our project aims to develop a mobile application that can recognize coffee machines in real-time using a smartphone camera and provide voice instructions to guide users through the coffee-making process. The application is designed to be accessible for all users with visual impairments.

Project Objectives

  • Develop a system for accurate and rapid real-time recognition of coffee machines via smartphone cameras.
  • Initially focus on recognizing a specific coffee machine model with high accuracy.
  • Provide real-time audio feedback to guide users through the coffee-making process.
  • Create an application that is accessible and optimized for visually impaired users.
  • Establish a solid foundation for future research and enhancements.
